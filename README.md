<h1>Squeezebox</h1>
<h2>Max compression. Max sharing. Max savings.</h2>

<p>Squeezebox is a compression tool that allows you dramatically shrink files to make it easier and faster to send to friends,
upload to socials, or just save space! </p>

<b>Please Note:</b>
<p>This is a full working model. There will be a web interface available in the near future.</p>
